# Журнал изменений — Crossout Session Overlay

## Версия 1.0 (текущая)

**Дата:** 2025

### Основной функционал

- Оверлей статистики сессии Crossout для трансляции на Twitch (тип: Video Overlay / Видео: на весь экран).
- Отображение в реальном времени: сводка сессии (победы/поражения, K/D, средний урон, рейтинг), история боёв, рейтинг по игрокам и оружию.
- Данные передаются с ПК стримера программой ExOut Logger на backend (Vercel) и отображаются зрителям поверх видео.
- Поддержка режима «Тест на сервере» (Hosted Test): статическая сборка оверлея без inline-скриптов (CSP), запросы к backend по CORS с домена расширения.

### Интерфейс оверлея

- Вкладки «История» и «Рейтинг».
- В истории: список боёв с картой, режимом, результатом (победа/поражение), убийствами, смертями, уроном; при клике — состав команд и детали игроков.
- В рейтинге: таблицы по игрокам и по оружию, поиск по названию оружия.
- При выборе игрока: подсветка игроков, по которым был нанесён урон или которые были убиты выбранным игроком; отображение среднего урона по выбранному оружию в заголовке.
- Иконки для ботов и игроков в группе (party) в таблицах игроков.
- Тёмная тема, вертикальная и горизонтальная прокрутка контента, стилизованные полосы прокрутки.

### Технические изменения

- Backend: API сессий (`/api/session/[channelId]`, `/api/session/update`), хранение сессий в Vercel KV (Redis).
- CORS: разрешены запросы с доменов `*.ext-twitch.tv` и localhost для доступа к API с оверлея в Hosted Test.
- Оверлей для Hosted Test собирается через Vite (отдельная сборка без Next.js runtime), в HTML только внешние скрипты — соответствие политике CSP Twitch.
- В консоли расширения необходимо добавить домен backend в «Белый список доменов для получения URL» (Capabilities) для работы в режиме «Тест на сервере».

### Зависимости для стримера

- Программа **ExOut Logger** для парсинга логов Crossout и отправки данных на backend.
- В настройках ExOut Logger: Backend URL и числовой Twitch Channel ID; при использовании расширения в Hosted Test backend должен быть добавлен в CSP-белый список расширения.
